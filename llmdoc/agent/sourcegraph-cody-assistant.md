# Sourcegraph Cody: AI Assistant with Codebase Context (2025)

## Overview

Sourcegraph Cody is an AI coding assistant leveraging advanced search and comprehensive codebase context to help developers understand, write, and fix code. It integrates with multiple IDEs and provides context-aware completions, chat, and custom prompt capabilities.

## Pricing Plans (2025)

### Free Plan
- Unlimited autocompletion suggestions per month
- Whole and multi-line suggestions included
- 200 chats/prompts per month
- Custom prompt creation capability
- Uses DeepSeek-Coder-V2 model for autocomplete
- Uses Claude 3.5 Sonnet model for chat and prompts

### Cody Pro
- $9/user/month
- Extended context window up to 1 million tokens
- Priority access to new features
- Enhanced AI models for faster, smarter responses
- Email support included

### Enterprise Starter
- $19 per user/month
- For growing organizations
- Supports up to 50 developers
- Advanced AI and search capabilities

### Enterprise
- $59 per user/month
- Comprehensive AI and search features
- Enterprise-level security
- Scalability for teams of 25+ developers
- Advanced flexibility

### Chat Interface
- Conversational interaction with codebase
- Ask questions about code
- Generate code snippets

- Receive explanations
- Utilizes context from open files and repositories

### Autocompletion

- Multi-line suggestions

### Codebase Understanding
- Entire codebase analysis
- Deeper contextual awareness for smarter suggestions
- Enhanced autocompletions from codebase knowledge

- JetBrains IDEs integration

- Multiple IDE support

### Model Support
- Claude 3.5 Sonnet for chat and prompts

### Custom Prompts
- Create custom prompts for specific tasks

## Performance Metrics

### Claude 3 Sonnet Adoption
- Approximately 75% increase in code insert rate after Claude 3 Sonnet deployment
- Users accept almost twice as much code from Cody suggestions
- Significant improvement in suggestion quality and relevance

## Strengths

- Free plan with 200 chats/month provides useful starter experience
- Cody Pro pricing at $9/month is competitive
- 1 million token context window in Pro plan enables deep understanding
- Multiple IDE support across major platforms
- Claude 3.5 Sonnet model provides high-quality suggestions
- Custom prompts for workflow optimization
- Strong performance metrics showing user adoption
- Enterprise-grade security for large teams

## Weaknesses and Limitations

- Free tier limited to 200 chats/prompts per month
- Enterprise plans require custom negotiation
- Smaller ecosystem compared to GitHub Copilot
- Less IDE coverage compared to some competitors
- No dedicated agent mode like Cursor or Windsurf
- Free plan uses different model (DeepSeek) than premium
- Custom pricing for Enterprise makes budgeting difficult
