# GitHub Copilot: Overview, Features, and Pricing (2025)

## Overview

GitHub Copilot is an AI coding assistant developed by GitHub (Microsoft) that provides code completion, chat assistance, and agentic capabilities throughout the software development lifecycle. It operates as a suggestion layer integrated into developers' existing IDEs and development environments.

## Pricing Plans (2025)

GitHub Copilot offers five distinct tiers:

- **Free**: $0/month - 2,000 code completions and 50 premium requests per month. Access to almost all Copilot features using base AI models like GPT 4.1, o3-mini, and Gemini 2.0
- **Copilot Pro**: $10/month - Unlimited completions, 300 premium requests per month, access to premium models in Copilot Chat, access to Copilot coding agent
- **Copilot Pro+**: $39/month - 1,500 premium requests per month, access to all AI models including Claude Opus 4 and OpenAI o3
- **Copilot Business**: $19/user/month - IP indemnity, centralized management, audit logs, features in IDE, CLI, and GitHub Mobile
- **Copilot Enterprise**: $39/user/month - 1,000 premium requests, GitHub.com Chat integration, knowledge bases, and custom models trained on organization's codebase

## Key Features and Capabilities

### Code Completions
- Contextualized code suggestions in IDEs
- Inline code completion as developers type
- Multi-file awareness in code suggestions

### Copilot Chat
- Conversational assistance within the IDE
- Code explanations and documentation
- Chat available in GitHub and IDEs

### Copilot Agent
- Autonomous handling of feature development
- Bug fixes and pull request management
- Coding agent functionality for complex tasks

### Copilot CLI
- Command-line interface support
- Integration with development workflows

### Premium Requests System
- Chat, agent mode, code review, and CLI features use premium requests
- Extra requests beyond plan limit cost $0.04 each
- Premium model access requires higher tier subscriptions

## IDE and Platform Integration

- GitHub.com integration
- IDE support (VS Code, JetBrains, Visual Studio)
- GitHub Mobile support
- CLI support

## Model Access

- **Free Plan**: Base models (GPT 4.1, o3-mini, Gemini 2.0)
- **Pro/Pro+ Plans**: Premium models including Claude Opus 4, OpenAI o3
- **Enterprise Plans**: Custom models trained on organization's codebase

## Enterprise Features

- Centralized management and seat allocation
- IP indemnity protection
- Audit logs and compliance features
- Custom models and knowledge bases for organizational context

## Technical Implementation

- Contextualized assistance throughout software development lifecycle
- Integration with code repositories and existing workflows
- Support for code explanations and documentation answers within GitHub

## Strengths

- Lowest entry price point with free and $10/month Pro option
- Generous free tier with 2,000 completions and 50 premium requests
- Deep GitHub integration and ecosystem alignment
- Strong agent mode capabilities for autonomous task handling
- Flexible model selection with access to multiple providers' models
- Well-established market presence with mature feature set

## Weaknesses and Limitations

- Premium requests system may feel restrictive for high-volume users
- Agent mode requires premium tier subscription
- Rate limiting on free tier may limit practical use for some workflows
- Less deep codebase understanding compared to specialized IDE-only tools
