# JetBrains AI Assistant: IDE Integration and Capabilities (2025)

## Overview

JetBrains AI Assistant is integrated across JetBrains IDE family (IntelliJ IDEA, PyCharm, WebStorm, etc.), providing code completion, intelligent chat, multi-file editing, and support for multiple AI models with advanced context management.

## Pricing and Availability

### Free Tier
- Unlimited code completion
- Powerful local workflows
- Limited cloud-based features
- Available in all JetBrains IDEs

### Premium Features
- Requires subscription to JetBrains IDE (not separate purchase)
- Included in IDE subscription plans
- No additional cost beyond IDE subscription

## Key Features and Capabilities

### Code Completion
- Single-line autocomplete adhering to coding style
- Entire block code suggestions
- Naming convention compliance
- Context-aware recommendations

### AI Chat with Agent Mode
- Chat interface with supported language models
- Model switching capabilities
- Agent mode for complex activities
- Autonomous implementation of fixes
- Code refactoring automation
- Test generation
- Advanced multi-step task handling

## Major 2025 Updates

### Multi-File Editing
- Chat-based multi-file edit suggestions
- Changes span across entire projects
- Retrieval-Augmented Generation (RAG) locates relevant files
- Single interaction reduces repetitive tasks

### Image Support
- Image attachments in chat
- Supported for Anthropic and OpenAI models
- Error diagnosis from screenshots
- Missing class generation from diagrams
- Architecture understanding through visual documentation

### Enhanced Context Management
- Advanced RAG for project-wide understanding
- Surfaces relevant files, methods, and classes
- Context includes recently accessed files
- Deeper project comprehension

### Model Selection
- Automatic model selection for optimal performance
- Cost tier indicators
- Reasoning capability indicators
- Beta/Experimental status visibility
- Visual indicators for model capabilities

### Additional Capabilities

**Next Edit Suggestions**:
- Suggests possible edits throughout file
- Quick application of suggestions
- Sequential navigation through needed changes

**Custom Local Models**:
- Third-party local model setup
- Custom code completion models
- Custom inline code generation
- Custom commit message generation

**MCP Integration**:
- Model Context Protocol support
- Secure access to external data sources
- Integration with third-party tools
- Connection to file systems and APIs
- Extended context from additional sources

**Web Command**:
- /web command for documentation lookup
- Fetches troubleshooting tips
- Retrieves technical resources
- Direct integration in chat window

## Strengths

- Free unlimited code completion in all IDEs
- Deep IDE-level integration providing superior context
- Multi-file editing with intelligent file discovery
- Image support for visual problem-solving
- Advanced MCP integration for extensibility
- Local model support for offline capability
- Web integration for documentation access
- No additional cost beyond IDE subscription

## Weaknesses and Limitations

- Cloud-based features limited on free tier
- Requires JetBrains IDE subscription (not standalone)
- Smaller market presence for standalone AI features compared to GitHub Copilot
- IDE subscription required increases total cost
- Image support limited to Anthropic and OpenAI models
- Smaller ecosystem for third-party integrations compared to VSCode
