# Claude Anthropic Code Assistant: SDK and API Capabilities (2025)

## Overview

Claude Code is an agentic coding assistant by Anthropic designed specifically for software development tasks. Available as a web application and SDK, it enables developers to build and manage AI coding agents for tasks like feature development, bug fixing, and codebase navigation.

## Claude Code Availability

- **Web Application**: Launched in October 2025, accessible through browser
- **IDE Integration**: Claude Agent available in JetBrains IDEs
- **SDK**: TypeScript, Python, and command-line tools
- **Subscription Access**: Available to Claude Pro ($20/month), Max ($100/month), and Team plan subscribers

## API Capabilities and Features (2025)

### Code Execution Tool
- Runs Python code in sandboxed environment
- Produces computational results and data visualizations
- Transforms Claude from code-writing assistant to data analyst
- Iterates on visualizations and cleans datasets within API calls
- Generates insights directly within API requests

### MCP Connector (Model Context Protocol)
- Connects Claude to remote MCP servers without writing client code
- Automatic connection management and tool discovery
- Error handling managed automatically by API
- Access powerful third-party tools by adding remote MCP server URL
- Immediate access to diverse tool ecosystems

### Files API
- Simplified document storage and access
- Upload documents once and reference repeatedly
- Reduces file management overhead in every request
- Streamlines document handling in long-running workflows

### Extended Prompt Caching
- Standard 5-minute time to live (TTL) for cached prompts
- Extended 1-hour TTL option - 12x improvement
- Reduces costs for long-running agent workflows
- Enables efficient reuse of large context windows

## Context Window

- Claude Sonnet 4 model with 1 million token context window
- Handles requests up to 750,000 words
- Can process entire "Lord of the Rings" trilogy or 75,000+ lines of code
- Extensive context enables deeper codebase understanding

## Pricing and Access
