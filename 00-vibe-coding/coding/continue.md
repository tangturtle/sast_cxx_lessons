# Continue: Comprehensive Documentation (2025)

## Overview

Continue is an open-source AI code assistant extension for VS Code and JetBrains IDEs that prioritizes flexibility, privacy, and developer control. Unlike proprietary solutions, Continue allows developers to choose their own AI models, run models locally, and maintain complete control over their code and data. The platform is built on the principle that developers should have freedom in selecting their AI tools and models, making it ideal for privacy-conscious developers and organizations with specific requirements.

### Market Position

Continue has positioned itself as the open-source, developer-friendly alternative:

- **Open source**: Fully transparent and community-driven
- **Model flexibility**: Support for any LLM (Claude, GPT-4, Gemini, local models)
- **Privacy-first**: Run models locally or use your own API keys
- **Developer control**: Complete control over configuration and data
- **Free**: No subscription required, only pay for model usage
- **Extensible**: Highly customizable for specific workflows

## Key Features and Capabilities

### 1. Model Flexibility

- **Multiple LLM support**: Claude, GPT-4, Gemini, Llama, Mistral, etc.
- **Local models**: Run models on your machine
- **Self-hosted**: Deploy your own model server
- **Model switching**: Easily switch between models
- **Custom models**: Support for custom fine-tuned models

### 2. AI Chat

- **IDE-integrated chat**: Chat panel within VS Code/JetBrains
- **Code context**: Automatically includes relevant code
- **File references**: Reference specific files in chat
- **Code generation**: Generate code from descriptions
- **Explanations**: Ask questions about code

### 3. Code Completions

- **Inline suggestions**: Real-time code suggestions
- **Multi-file awareness**: Understands project context
- **Language support**: Works with all major languages
- **Customizable**: Configure completion behavior
- **Performance**: Optimized for responsiveness

### 4. Privacy and Control

- **Local execution**: Run models on your machine
- **No data collection**: Your code stays private
- **API key control**: Use your own API keys
- **Self-hosted**: Deploy on your infrastructure
- **Offline support**: Works without internet connection

### 5. Extensibility

- **Custom commands**: Create custom AI commands
- **Plugins**: Extend functionality with plugins
- **Configuration**: Highly customizable configuration
- **Integrations**: Integrate with your tools
- **Open source**: Modify and extend as needed
