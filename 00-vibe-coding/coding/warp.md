# Warp: Agentic Development Terminal (2025)

## Overview

Warp is a modern, GPU-accelerated terminal that embeds AI for command suggestions, explanations, and agentic workflows (from writing code to deploying). It brings context from your repo and shell history to help automate terminal tasks.

- Website: <https://www.warp.dev/>
- AI: <https://www.warp.dev/warp-ai>

## Key Features

- AI command search and natural-language prompts
- Explain output and errors; generate shell scripts
- Command blocks with rich UI and sharing
- Agent workflows spanning dev → test → deploy
- Team features for shared workflows

## Strengths

- Fast native terminal with great UX
- Practical AI for shell tasks and onboarding
- Free AI quota for light use

## Limitations

- Works best on macOS/Linux; Windows via WSL
- AI features limited under free quota

## Pricing/Access (2025)

- Warp AI: free up to 100 requests/user/month (per site)
- Team/Enterprise plans for higher limits and collaboration

## Use Cases

- Learn complex CLI tools faster
- Automate repetitive shell workflows
- Explain and remediate build/test failures

## Getting Started

1. Install Warp from website
2. Enable Warp AI and sign in
3. Use Cmd-K to ask for commands or explanations

## Comparison Points

- vs Aider/Cline: Warp is a terminal with AI helpers; not a code-editing agent
- vs VS Code Terminal + extensions: Warp offers rich blocks and native UX
