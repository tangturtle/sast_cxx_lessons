# 通义灵码 (Tongyi Lingma): Comprehensive Guide (2025)

## 概述 Overview

通义灵码是阿里云推出的 AI 编程助手，支持 VS Code/JetBrains 等主流 IDE，提供代码补全、解释、重构、文档/测试生成等能力，并针对中文开发者场景做了深度优化。

- 插件市场: <https://marketplace.visualstudio.com/items?itemName=Alibaba-Cloud.tongyi-lingma>
- 安装指南: <https://www.alibabacloud.com/help/en/lingma/user-guide/download-the-installation-guide>

## 主要特性 Key Features

- IDE 内联补全与聊天
- 多语言支持（Java/JS/TS/Python/Go 等）
- 代码解释与重构建议
- 测试/注释/文档生成
- 企业级隐私与合规控制（阿里云生态）

## 优势 Strengths

- 面向中文生态与企业场景优化
- 与阿里云产品/控制台集成更顺滑
- 易用的插件与中文文档

## 限制 Limitations

- 海外可用性与速度因区域而异
- 自主智能体能力不如 Cursor/Windsurf 类工具

## 使用场景 Use Cases

- 日常补全与解释
- 基于阿里云的项目开发
- 中文代码/文档生成

## 定价/获取 Pricing/Access

- 插件免费安装
- 具体计费与配额依产品与地区而异（请参考官方文档/控制台）

## 集成 Integration

- VS Code、JetBrains 系列

## 快速开始 Getting Started

1. 在 VS Code/JetBrains 安装通义灵码插件
2. 登录阿里云账号并完成激活
3. 在项目中直接试用补全与聊天

## 对比 Comparison Points

- vs Copilot：中文生态与阿里云集成更好
- vs Cursor：Cursor 是 AI IDE，灵码是 IDE 助手；灵码更轻量
