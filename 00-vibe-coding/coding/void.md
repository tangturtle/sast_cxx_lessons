# Void

## 概览（Overview）

Void 是一个开源、AI 增强的代码编辑器项目（通常被描述为 Cursor 的开源替代思路）。社区资料显示其目标是把代理式工作流、本地/远程模型与现代编辑体验结合起来。由于生态演进，项目活跃度可能阶段性波动；请以官方仓库为准。

## 核心能力（Key Features）

- 开源可扩展：社区可贡献功能、主题与扩展。
- 代理式工作流：对话、任务拆解、自动编辑/补丁生成（以实际实现为准）。
- 多模型/本地推理：可对接本地/云端模型（取决于插件与配置）。
- 开发者体验：快捷命令、命令面板、代码搜索等现代编辑特性。

## 适用场景（Use Cases）

- 偏好开源栈、希望二次开发或自建的团队/个人。
- 在离线/隐私环境下对接本地模型的实验与落地。
- 用作 IDE 主体或与 VS Code/JetBrains 并行试验。

## 优势（Strengths）

- 开源透明、可分叉改造。
- 学习/研究 AI IDE 架构的良好载体。

## 局限（Limitations）

- 项目活跃度与功能完备度需以仓库为准，可能不及成熟商用产品。
- 集成深度（Git/CI/测试/安全）与生态配套（插件/团队功能）待评估。

## 定价/获取（Pricing / Access）

- 开源仓库：请参考 GitHub（以最新为准）。

## 对比（Comparison）

- vs Cursor/Windsurf/Zed：成熟度与生态尚有差距，但开源/可控性更强。
- vs 纯终端代理：Void 更偏 IDE 一体化体验。
