# Zed: Comprehensive Documentation (2025)

## Overview

Zed is a next-generation code editor built from scratch in Rust, designed for high-performance collaboration with both humans and AI. Unlike editors built on Electron, Zed achieves exceptional performance with 120fps rendering and minimal latency. The editor combines native AI capabilities with collaborative features, making it ideal for developers who want a modern, fast editor with integrated AI assistance. Zed represents a fresh approach to code editing, prioritizing performance and developer experience.

### Market Position

Zed has positioned itself as the high-performance alternative to VS Code:

- **Performance-first**: 120fps rendering, minimal latency
- **Native AI integration**: Built-in AI features without extensions
- **Collaborative editing**: Real-time collaboration capabilities
- **Modern architecture**: Built in Rust for speed and reliability
- **Free editor**: Core editor is free; AI features are token-based

## Key Features and Capabilities

### 1. High-Performance Editor

- **120fps rendering**: Smooth, responsive interface
- **Minimal latency**: Instant response to user input
- **Rust-based**: Built for speed and reliability
- **Low memory usage**: Efficient resource utilization
- **Cross-platform**: macOS, Linux, and Windows support

### 2. Native AI Integration

- **Built-in AI**: No extensions needed for AI features
- **Multiple models**: Support for Claude, GPT-4, Gemini
- **Code completions**: Intelligent inline suggestions
- **AI chat**: Conversational assistance within editor
- **Token-based pricing**: Pay only for what you use

### 3. Collaborative Editing

- **Real-time collaboration**: Multiple developers editing simultaneously
- **Presence awareness**: See where other developers are working
- **Shared cursors**: Track other developers' positions
- **Conflict resolution**: Intelligent handling of concurrent edits
- **Session management**: Easy setup and management of collaborative sessions

### 4. Code Intelligence

- **Language support**: Comprehensive support for major languages
- **Syntax highlighting**: Accurate and performant highlighting
- **Code navigation**: Quick navigation and search
- **Refactoring**: Intelligent code refactoring tools
- **Debugging**: Integrated debugging capabilities

### 5. Extension Ecosystem

- **Extension support**: Extensible through plugins
- **VS Code compatibility**: Many VS Code extensions work in Zed
- **Custom extensions**: Create custom extensions for specific needs
- **Theme support**: Customize editor appearance

### 6. Developer Experience

- **Command palette**: Quick access to commands
- **Keyboard shortcuts**: Customizable keyboard bindings
- **Settings**: Flexible configuration options
- **Terminal integration**: Built-in terminal support
- **Git integration**: Native Git support
