# Kilo Code: Comprehensive Guide (2025)

## Overview

Kilo Code is an open-source AI coding agent for VS Code (and JetBrains support in progress). It helps plan, build, and fix code via autonomous agents integrated in your editor. Kilo supports multiple providers and emphasizes transparency and control.

- Website: <https://kilocode.ai/>
- VS Code: <https://marketplace.visualstudio.com/items?itemName=kilocode.Kilo-Code>
- Docs: <https://kilocode.ai/docs/getting-started/installing>

## Key Features

- Agentic plans with file editing and shell commands
- Compatible with OpenAI, Anthropic, Grok, Gemini, and more
- Workspace rules and prompts to align with your style
- Diff review and approval checkpoints
- Community-driven, open-source

## Strengths

- Open-source and extensible
- Clear agent plans and review gates
- Multi-model flexibility and provider toggles

## Limitations

- Feature set still evolving vs. Cursor/Windsurf
- Quality highly model-dependent
- JetBrains support not as mature as VS Code

## Use Cases

- Implement features with tests
- Fix failing pipelines and linters
- Refactor modules and update imports

## Pricing/Access

- Free extension; BYO API keys
- Some community-hosted options exist; check docs

## Setup

1. Install from VS Code Marketplace
2. Configure preferred provider(s) in settings
3. Start a task; review plan and diffs

## Comparison Points

- vs Cline/Roo: Similar autonomy layer; Kilo emphasizes open-source and provider toggles
- vs Copilot: Kilo is agentic (task-level), not just inline completions
