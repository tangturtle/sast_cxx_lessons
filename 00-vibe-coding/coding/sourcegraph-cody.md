# Sourcegraph Cody: Comprehensive Documentation (2025)

## Overview

Sourcegraph Cody is an AI coding assistant built on top of Sourcegraph's powerful code search and intelligence platform. Cody uniquely combines AI assistance with deep codebase understanding through Sourcegraph's code graph technology, enabling context-aware coding assistance that understands your entire repository structure. The platform is particularly strong for large-scale codebases and organizations that need sophisticated code search capabilities alongside AI assistance.

### Market Position

Cody has established itself as the AI assistant for developers working with large, complex codebases:

- **Code graph integration**: Leverages Sourcegraph's code intelligence
- **Massive context windows**: Up to 1 million tokens with Gemini 1.5 Flash
- **Multi-repository support**: Understands relationships across repositories
- **Enterprise focus**: Strong adoption in large organizations
- **Code search integration**: Combines AI with powerful code search

## Pricing Plans (2025)

### Free Plan

- **Cost**: $0/month
- **Features**: Basic AI chat and code completions
- **Best for**: Individual developers and open-source projects
- **Limitations**: Limited context window, basic features only
- **Model**: Access to limited model options

### Pro Plan

## Key Features and Capabilities

### 1. Code Graph Integration

- **Codebase understanding**: Deep analysis of code structure and relationships
- **Cross-file context**: Understands dependencies across files
- **Symbol navigation**: Intelligent code navigation and search
- **Dependency tracking**: Understands how code components interact
- **Architecture awareness**: Recognizes architectural patterns

### 2. Expanded Context Windows

- **Gemini 1.5 Flash**: 1 million token context window
- **Claude 3.5 Sonnet**: 200,000 token context window
- **GPT-4o**: 128,000 token context window
- **Dynamic context**: Automatically selects optimal context for tasks
- **Pull request context**: Can review entire PRs with full context

### 3. AI Chat and Assistance

- **Code explanation**: Understand complex code sections
- **Documentation generation**: Automatic documentation creation
- **Refactoring suggestions**: Intelligent code improvement recommendations
- **Bug detection**: Identifies potential issues
- **Test generation**: Creates unit tests automatically

### 4. Multi-Repository Support

- **Cross-repo search**: Search and understand code across repositories
- **Dependency analysis**: Understand relationships between repos
- **Monorepo support**: Handles large monorepo structures
- **Organization-wide context**: Understands entire codebase organization

### 5. Code Search Integration

- **Powerful search**: Leverage Sourcegraph's code search capabilities
- **Semantic search**: Find code by meaning, not just keywords
- **Regex support**: Advanced pattern matching
- **Language-aware**: Understands code structure and syntax

### 6. IDE Integration

- **VS Code extension**: Full-featured VS Code integration
- **JetBrains support**: IntelliJ and other JetBrains IDEs
- **Web interface**: Browser-based access
- **CLI support**: Command-line integration
