# Same.dev / Same.new

## 概览（Overview）

Same（域名同名为 same.dev / same.new）是一款“对话式建站/应用生成”平台，强调通过与 AI 聊天快速构建网站/轻应用原型，并提供可视化编辑和持续迭代能力。官方主域通常为 same.new，社区也以 Same.dev 指代其产品定位。

注：该产品快速迭代，本文总结公开材料中的通用能力；具体功能以官网为准。

## 核心能力（Key Features）

- 对话式创建：用自然语言描述目标，快速生成初版站点/页面/组件。
- 可视化编辑：在生成结果基础上进行拖拽/属性配置/样式微调。
- 组件与模板：常见页面/区块/布局模板复用，降低从零开始的成本。
- 导出与托管：支持导出代码或在平台托管（以官方能力为准）。
- 多模型/智能体：按场景选择不同模型与“代理”策略（以官方实现为准）。

## 适用场景（Use Cases）

- 营销落地页/活动页的快速搭建与迭代。
- MVP 原型验证与 A/B 文案/样式实验。
- 个人/团队主页、博客、作品集等轻型站点。

## 优势（Strengths）

- 上手极快：从对话到页面可见成果的路径很短。
- 交互友好：“生成→调整→再次生成”的闭环顺畅。
- 低门槛：面向非工程背景用户也较友好。

## 局限（Limitations）

- 定制深度：高度定制/复杂交互仍可能需要工程介入。
- 可移植性：导出代码质量、二次开发体验需结合版本评估。
- 可观测性：对生成逻辑/设计决策的可追踪程度有限。

## 对比（Comparison）

- vs Bolt.new / v0.dev：Same 更偏“聊天生成 + 可视化微调”；Bolt/v0 更偏“从文字到可运行代码/React UI”。
- vs Lovable：定位接近，具体在导出能力与应用复杂度支持上有差异。
