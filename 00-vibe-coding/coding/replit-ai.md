# Replit AI Agent / Ghostwriter: Comprehensive Guide (2025)

## Overview

Replit integrates AI across its browser-based IDE and hosting platform. The AI Agent (formerly Ghostwriter features) can build apps from natural language, generate/edit code, explain errors, and help deploy—all within Replit workspaces.

- Site: <https://replit.com/>
- AI Landing: <https://replit.com/ai>
- Pricing: <https://replit.com/pricing>

## Key Features

- Chat to build apps and components
- Inline code completions and fix suggestions
- Project scaffolding and deployment integration
- Multi-language support in the browser
- Collaborative editing and sharing

## Strengths

- Zero-setup browser IDE with hosting
- Great for teaching, prototyping, and demos
- Tight integration from code → run → deploy

## Limitations

- Power users may prefer local IDEs for large repos
- AI agent autonomy has quotas and costs

## Pricing/Access (2025)

- Plans vary; see pricing page for AI/Agent quotas
- Free tier with limited AI; paid tiers unlock more autonomy

## Use Cases

- Prototype full-stack apps quickly
- Learn languages and frameworks interactively
- Hackathons, workshops, education

## Getting Started

1. Create a Replit account
2. Open a new Repl and enable AI
3. Chat with the Agent to scaffold and iterate

## Comparison Points

- vs v0/Lovable: Replit is a full IDE + hosting; v0/Lovable focus on app generation
- vs VS Code + Copilot: Replit is browser-based with integrated run/deploy
