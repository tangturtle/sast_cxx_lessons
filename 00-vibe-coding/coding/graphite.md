# Graphite: AI Code Review and PR Stacking (2025)

## Overview

Graphite is a developer workflow platform centered on stacked pull requests and fast code review. Its AI features (Graphite Agent, AI Reviews, and Diamond reviewer) provide context-aware review comments, risk detection, and suggestions that integrate with GitHub.

- Website: <https://graphite.dev/>
- AI Reviews: <https://graphite.dev/features/ai-reviews>
- Guide: <https://graphite.dev/guides/how-ai-code-review-works>

## Key Features

- AI-powered code review comments on PRs
- Risky change detection and suggested fixes
- Stack-aware review across dependent PRs
- GitHub integration with status checks
- Suggested test changes and refactor hints

## Strengths

- High-signal review vs. generic LLM reviewers
- Deep integration with PR stacks and GitHub
- Speeds up review cycles for larger teams

## Limitations

- Focused on review; not a coding agent/IDE
- Requires GitHub app/permissions
- Pricing primarily targets teams/enterprises

## Use Cases

- Accelerate PR review throughput
- Catch subtle issues earlier in the cycle
- Improve review consistency across teams

## Pricing/Access

- Team and Business plans; see website for current pricing
- GitHub installation required

## Integration

- GitHub app, status checks, and CI integrations
- Works alongside Copilot/Cline/etc. for authoring

## Getting Started

1. Install Graphite GitHub app
2. Enable AI Reviews on repos/teams
3. Configure review policies and thresholds

## Comparison Points

- vs GitHub Copilot PR: Graphite specializes in PR review depth; Copilot also offers PR suggestions but is more general
- vs Sonar/Static tools: Graphite is LLM-first and contextual; static tools are rule-based
