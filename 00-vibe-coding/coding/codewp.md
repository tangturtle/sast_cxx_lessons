# CodeWP（状态：已下线/迁移中）

## 概览（Overview）

CodeWP 曾是一款面向 WordPress 生态的 AI 代码生成与片段助手，支持 PHP/JS、WooCommerce、Gutenberg/Elementor 等。然而根据 2024‑12 的官方公告，其所属的 WPAI 被 Automattic 收购，CodeWP 与 AgentWP 以“当前形态”停止服务/将被替代，用户与功能正向新的产品线迁移。

当前（2025‑11）建议关注 WordPress 官方/Automattic 的 AI 能力与插件生态作为替代路径。

## 历史能力（Historical Features）

- 面向 WP 的片段生成（actions/filters/shortcode）、调试与解释。
- 针对 WooCommerce/Elementor/Gutenberg 的专用模板与提示语集。
- 与站点联动（通过插件/密钥）直接推送/测试片段。

## 现状与替代（Status & Alternatives）

- 状态：产品线已被收购并宣布停用原形态；用户迁移参考官方公告。
- 替代方向：
  - WordPress/Automattic 侧的 AI 能力/插件（例如官方 WP AI 方案）。
  - 通用 IDE 助手（Copilot/Continue/Cody/Claude Code 等）。
  - 终端代理（Aider/Goose/OpenCode）配合 WP 项目使用。

## 风险与注意（Notes）

- 互联网上仍有旧文章/教程，但接口与站点可能已不可用或行为变化，请以最新公告为准。
