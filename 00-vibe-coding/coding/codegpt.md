# CodeGPT (VS Code): Comprehensive Guide (2025)

## Overview

CodeGPT is a popular AI assistant extension for VS Code and JetBrains that provides chat, code generation, refactoring, debugging help, and documentation generation. It supports BYO API keys (OpenAI, Anthropic, Google) or the CodeGPT service, and includes an optional Code Interpreter sandbox.

- Website: <https://codegpt.co/>
- Docs: <https://docs.codegpt.co/>
- VS Code: <https://marketplace.visualstudio.com/items?itemName=DanielSanMedium.dscodegpt>

## Key Features

- In-editor chat with code context
- Inline explain/fix/refactor
- Test and doc generation
- Commands palette integration (/explain, /test, /doc, etc.)
- Optional Code Interpreter for running snippets
- BYO key or CodeGPT-managed API

## Strengths

- Mature UX; easy to adopt
- Works with multiple providers and models
- Good presets and prompt commands

## Limitations

- Autonomy is limited vs. agentic tools (Cline/Roo/Kilo)
- Quality depends on chosen model
- Interpreter runs are sandboxed with limits

## Use Cases

- Quickly explain unfamiliar code
- Generate tests for a function
- Refactor small sections safely
- Draft documentation and comments

## Integration

- VS Code, JetBrains plugins

## Getting Started

1. Install CodeGPT from Marketplace
2. Configure model provider (OpenAI/Anthropic/Google or CodeGPT)
3. Use the chat panel or context menus to trigger actions

## Comparison Points

- vs Copilot: CodeGPT centers on chat/commands; Copilot excels at inline completions
- vs Continue: Continue is open-source and highly configurable; CodeGPT offers a managed experience with presets
